# Logo Files

This directory contains the branding assets for Channels DVR Collection Manager.

## Files

- **logo.png** - Main logo (1024x1024px) - Used in header and documentation
- **favicon.png** - Browser tab icon

## Design

The logo features:
- **Folder with Collections** - Organized content management
- **Spreadsheet Grid** - Structured data and lists
- **Magnifying Glass with Checkmark** - Verification, search, and validation
- **Play Buttons** - Media and video content
- **Color Scheme**: Purple gradient background - Modern, tech-focused
- **Text**: "ChannelsDVR Collection Manager"

## Usage

The PNG logo is used throughout the application:
- Header logo (50x50px display size with rounded corners)
- Browser favicon
- Documentation and README images
- Social media previews
- GitHub repository image
